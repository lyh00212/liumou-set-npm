## 这是一个镜像源切换库

### 1、安装

```bash
    npm install liumou-set-npm
```

### 2、可用命令

① 查看版本

```bash
lyh -V
```

② 查看已配置镜像源

```bash
lyh ls
```

③ 设置当前使用的镜像源

```bash
lyh use
```

④ 查看当前镜像源

```bash
lyh current
```

⑤ 测试镜像源地址速度

```bash
lyh ping
```

⑥ 添加自定义镜像源

```bash
lyh add
```

⑦ 删除自定义镜像源

```bash
lyh delete
```

⑧ 重命名自定义镜像源

```bash
lyh rename
```

⑨ 编辑自定义镜像源

```bash
lyh edit
```
